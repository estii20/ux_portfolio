E-commerce Platform: Design & Development

Duration: 4 months
Role: UX Designer & Django Developer
Category: E-commerce

KidSwap: A Full-Stack E-commerce Solution

This project was a personal initiative where I took on the challenge of both UX design and full-stack development using Django. The goal was to create an intuitive e-commerce platform for parents to buy and sell children's clothing. I focused on a user-centered design and a scalable architecture to ensure a seamless shopping experience across all devices.

Challenge

Designing and developing an efficient product categorisation and filtering system for a user base with limited time.

Solution

I implemented an intuitive navigation structure and a smart filtering system, built with Django, to enhance the user experience.

Key Results

  • Successfully designed and developed a fully functional e-commerce platform.
  • Implemented a robust Django backend ensuring efficient data management and scalability.
  • Achieved a highly responsive and intuitive user interface across all devices.
