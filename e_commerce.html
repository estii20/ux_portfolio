E-commerce Platform Design

Duration: 4 months
Role: Lead UI/UX Designer
Category: E-commerce

Project Overview - KidSwap

The client needed an e-commerce platform with great user experience, increase over time conversion rates, and implement modern design patterns. The goal was to create an intuitive design for parents to purchase and resell children's clothing. The shopping experience needed to be able to scale across multiple device types

Challenge

Product categorisation and filtering system needed to be simple for time poor users

Solution

Implemented intuitive navigation and smart filtering system

Key Results

150% increase in conversion rate

45% decrease in cart abandonment

60% increase in mobile purchases
