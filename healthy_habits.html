Healthy Habits Coach App Design ✨

A personal mobile application design project aimed at helping adults develop and maintain healthy lifestyle habits through **intuitive UX/UI** and personalised guidance.

UX Research UI Design Mobile App Health Tech

Project Overview

The Challenge

Design a user-friendly app that helps **busy adults develop and sustain** healthy habits through personalised coaching and simple progress tracking, overcoming typical user drop-off.

The Solution

Created an intuitive, aesthetically pleasing app with AI-powered coaching, **customisable meal plans**, workout routines, and simplified habit tracking features.

Design Process

🔍 Research & Discovery

  • Conducted **20 user interviews** to identify pain points.
  • Analysed competitor apps for feature gaps and best practices.
  • Created user personas and defined core feature sets.

🎨 Design & Implementation

  • Created low and high-fidelity wireframes in Figma.
  • Developed interactive prototypes for user flow testing.
  • Refined the UI based on **usability testing feedback**.

Core Product Features

AI Coach

Personalised daily recommendations and real-time feedback to guide habit formation.

Meal Planning

Customisable meal plans and straightforward nutrition logging and tracking.

Workout Library

Access to video-guided exercises, goal tracking, and progress visualisations.

Key Design Outcomes

Validated Design

User testing provided overwhelmingly positive feedback on the intuitive interface and habit-building features.

Sustained Engagement

Designed a compelling, rewarding user journey intended to foster long-term adherence to healthy habits.

Scalable Solution

Developed a robust, modular UI/UX framework ready for future feature development and platform expansion.
